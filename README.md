# Flight Scanner

A flight data analysis and booking simulation built with [Processing](https://processing.org/), using a 2,000-flight CSV dataset.

## Features

- **Flight Search** — search one-way or round-trip flights by route or date range, with autocomplete and input validation
- **Bookings** — view reservations and generate a digital boarding pass with barcode
- **Traffic Analytics** — compare on-time, delay, and cancellation rates across US regions and routes
- **Charts** — interactive pie and bar charts broken down by airline, airport, and flight status
- **Flight Map** — interactive USA map with colour-coded flight paths, clickable routes and airports

## Getting Started

1. Install [Processing 4](https://processing.org/download)
2. Clone the repo and open the main `.pde` sketch
3. Make sure the `data/` folder (CSV + SVG files) is in the sketch directory
4. Press **Run ▶**
